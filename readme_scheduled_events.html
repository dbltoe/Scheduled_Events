Scheduled Events for Zen Cart

Version 2.0.0 — requires Zen Cart v2.0.0 or later. Author: dbltoe.

Overview

Scheduled Events lets a store owner maintain a list of upcoming events (shows, fairs, expos, appearances, etc.) and display them on their own storefront page. Each event has a name, place, and start/stop dates; comments, booth location, an event information link, and driving directions are optional. Events are managed from Catalog > Scheduled Events in the admin.

Display can be further promoted via a sidebox — either as a link added into the current template's existing Information sidebox, or as the plugin's own sidebox (a simple link, or a rotating slider of currently qualifying events) placed via Design > Layout Boxes Controller like any other sidebox. All of this, along with the storefront page heading, field labels, and the display time window (how many days before an event's start date it becomes visible), is configurable under Configuration > Scheduled Events.

Installing, Upgrading, or Uninstalling

Initial Installation or Upgrade

Copy the contents of the zc_plugins/ScheduledEvents/v2.0.0 folder into your Zen Cart store's root directory, preserving the folder structure. Then, in the admin, go to Tools > Install/Update Plugins, locate Scheduled Events, and click Install (or Update, if upgrading from a prior version).

This creates the eventz database table (respecting your store's table prefix, if any), adds the Configuration > Scheduled Events settings group, and adds the Catalog > Scheduled Events admin menu entry.

Uninstalling

In the admin, go to Tools > Install/Update Plugins, locate Scheduled Events, and click Uninstall. This removes the eventz table and all of its data, the plugin's configuration settings, the admin menu entry, and any Layout Boxes Controller placement of its sidebox. Removing the plugin's files from the server afterward is optional but recommended.
